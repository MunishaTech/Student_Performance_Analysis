# Student Performance Data Analysis Project

## Project Overview
This project analyzes student academic performance data using Excel. 
The objective is to clean raw data, perform analysis, generate insights, 
and present summary reports using pivot tables and formulas.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Excel Formulas (IF, AVERAGE, COUNT, RANK, etc.)

## Dataset Description
The dataset contains student academic information including:
- Student Name
- Subject
- Marks
- City
- Performance Category
- Rankings
- Averages

## Project Workflow

1. Raw Data Collection  
   - Imported student data into Excel

2. Data Cleaning  
   - Removed blanks and errors  
   - Standardized names using TRIM and PROPER  
   - Converted data types  
   - Created derived columns like Pass/Fail and Above/Below Average

3. Data Analysis  
   - Subject-wise average marks  
   - City-wise student performance  
   - Top performers by subject  
   - Ranking students within each city  

4. Reporting & Visualization  
   - Created pivot tables  
   - Built summary dashboard  
   - Generated performance overview

## Key Insights
- Identified top performing subjects
- Found highest scoring students by subject
- Compared city-wise performance
- Ranked students based on marks
